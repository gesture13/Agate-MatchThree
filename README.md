# Match Three
Match Three, yaitu sebuah game casual seperti candy crush, dimana kita akan menghancurkan icon-icon yang muncul dengan cara membuat minimal 3 barisan icon yang sama.

## How To Install The Game
1. Go to [Standalone Folder] and download the zip.
2. Unzip and click Agate-MatchThree.exe
3. Click Play! to start the game
